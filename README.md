# AutoXML

Automação em Python para leitura de Notas Fiscais Eletrônicas (NFe) em formato XML e exportação dos dados em formato de tabela (Excel).

Este projeto tem como objetivo facilitar a extração de informações das NFs de forma automatizada, sem a necessidade de abrir cada arquivo manualmente.


## Funcionalidades

- Leitura automática de arquivos XML de uma pasta selecionada pelo usuário.  
- Extração dos campos especificados de cada nota.
- Gerar tabela em Excel com os dados.
- Compatível com Windows, sem necessidade de terminal (modo gráfico com seleção de pasta).

## Tecnologias utilizadas

- [Python 3](https://www.python.org/)
- [xmltodict](https://pypi.org/project/xmltodict/)
- [pandas](https://pandas.pydata.org/)
- [tkinter](https://docs.python.org/3/library/tkinter.html)
